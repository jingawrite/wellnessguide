# 이미지 파일 안내

이 폴더에 다음 이미지 파일들을 추가해주세요:

## 필요한 이미지 파일

### 1. `favicon.png`
- **용도**: 브라우저 탭에 표시되는 아이콘
- **권장 크기**: 32x32px 또는 64x64px (PNG 형식)
- **설명**: "웰"과 "ㄹ" 문자가 있는 둥근 사각형 아이콘

### 2. `og-image.png`
- **용도**: 소셜 미디어 공유 시 표시되는 이미지 (Open Graph)
- **권장 크기**: 1200x630px (PNG 형식)
- **설명**: "JINGA", "잘 사는 비결.", "웰니스 가이드" 텍스트가 있는 배너 이미지

## 파일 추가 방법

1. 이미지 파일을 준비합니다
2. 이 폴더(`public/`)에 다음 이름으로 저장합니다:
   - `favicon.png`
   - `og-image.png`
3. 파일이 추가되면 자동으로 적용됩니다

## 참고사항

- Vite는 `public` 폴더의 파일을 루트 경로(`/`)로 서빙합니다
- 이미지 파일은 절대 경로(`/favicon.png`, `/og-image.png`)로 참조됩니다
- 배포 시에도 동일하게 작동합니다
